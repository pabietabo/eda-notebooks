# EDA Notebooks
This repository contains exploratory data analysis (EDA) notebooks on real-world datasets.
Each notebook focuses on understanding the data, uncovering patterns, and generating insights that could inform downstream modeling or decision-making.

## Structure
notebooks/ – EDA notebooks  
datasets/ – data files (not tracked)  
images/ – exported plot

## Data

Datasets used in these notebooks are **not included** in this repository.

Each notebook contains a link to its data source and, where possible,
code to download the dataset automatically.

Downloaded datasets are expected to be placed in the `datasets/` folder.
